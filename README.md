## How to run locally

Follow the steps below to run locally.

1. Clone the repository
```
git clone https://github.com/360dialog/integrated-onboarding-demo
```

2. Install the dependencies
```bash
npm install
# or
yarn install
```

3. Run the local development environment
```bash
npm run dev
# or
yarn dev
```
